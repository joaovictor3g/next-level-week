# Backend da Aplicação Next Level Week 🚀 - Ecoleta ♼

### 🏃 How to run?
- Clone this repository with: `git clone`
- Open the folder backend.
- Run `npm run knex:migrate` to run all migrations
- Run `npm run dev`
- Application will be running on port `3333`
